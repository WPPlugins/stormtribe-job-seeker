== Description ==
-Add jobs for listing using custom post type
-By default area and career categories/taxonomies (more can be added)
-Easy to add with Widget, shortcodes or PHP coding
-Scrolling/rolling job list and/or static list
-Dynamic list jobs on pages based on area or career
-Extensive user customization (coding and css without breaking updates)
-Multisite/blog support
-Multilingual

Recommended minimum:
PHP 5.2 or later
WordPress 3.5 or later


== Installation ==
1.
Unzip the plugin to your wp-content/plugins/ directory.
Activate the plugin in wp-admin.

2.
Read the Information page on the plugin in wp-admin for further instructions.


== Upgrade Notice ==
1. Deactivate the plugin.
2. Unzip the new plugin to your wp-content/plugins/ directory. And replace every file.
3. Activate the plugin in wp-admin.
4. Check info page for any warnings.
